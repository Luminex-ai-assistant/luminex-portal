# Luminex Command Dashboard

A modern Kanban-style project management dashboard for tracking tasks, projects, and collaboration between Tag and Luminex.

## Features

- **Kanban Board**: Drag-and-drop task management across 5 columns (Backlog, To Do, In Progress, Review, Done)
- **Project Organization**: Tasks organized by project (Apartment Hunt, Airbnb, Immigration Work, Personal, Luminex Development)
- **Priority Levels**: P0 (Critical), P1 (High), P2 (Normal)
- **Assignee Tracking**: Tasks assigned to Tag or Luminex
- **Due Dates**: Visual indicators for overdue tasks
- **Filtering**: Filter by project, assignee, priority, and search
- **Responsive Design**: Works on desktop and mobile
- **Dark Theme**: Easy on the eyes for extended use

## Local Development

Simply open `index.html` in a web browser. No build step required.

## Deployment

This dashboard is designed to be deployed on Netlify with password protection enabled.

## Data Storage

Currently uses browser localStorage for task persistence. Google Sheets integration planned for future releases.
